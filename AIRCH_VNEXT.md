> Не переписывание. Эволюция изнутри.

---

## Суть сдвига

**AIR4 сейчас:**
```
Данные → Дашборд → Иногда говорит что-то умное
```

**vNext:**
```
Разговор → Понимание → Spaces → Говорит когда важно
```

AIR4 сейчас реагирует на данные. vNext реагирует на тебя.

---

## Главная идея

AIRCH — Personal Operating System. Не приложение которое открываешь.
Система которая всегда в контексте и говорит когда нужно.

Ощущение: Джарвис. Советник который знает тебя потому что у него данные.

**Три роли:**
- Знает — помнит контекст, историю, паттерны
- Говорит правду — называет вещи своими именами
- Помогает решить — конкретный следующий шаг

## 5 компонентов MVP (30 дней)

### 1. Identity Model
Таблица выводов о пользователе. Не факты — выводы.
```
"Ar4 принимает импульсивные решения под давлением дедлайна"
"В стрессе тратит больше на рестораны"
"Продуктивнее когда тренируется регулярно"
```
Новая таблица `identity_model` в SQLite. AIR4 заполняет автоматически из разговоров.

### 2. Spaces
Гибкие области жизни вместо фиксированных страниц.
Каждый Space = свой чат + своя память + свои рекомендации.

Стартовые Spaces:
- Financial Freedom
- Performance
- AIRCH (проект)
- Work

Space появляется по предложению AIR4 или по команде пользователя.

### 3. Morning Brief
Проактивное сообщение при первом открытии за день.
```
"Доброе утро. Тренировка груди — 4 дня назад.
До конца цикла 6 дней, свободных €340.
SkipMar не двигался 8 дней."
```
Коротко. Честно. Без воды.

### 4. Knowledge Layer
Слой между чатом и памятью. Хранит выводы, не данные.
```
Chat → Knowledge Layer → Identity Model
```
AIR4 автоматически создаёт и обновляет выводы после каждого разговора.

### 5. Follow-up Engine
AIR4 возвращается к важному.
- Сказал "встреча с Карлосом в пятницу" → после встречи спрашивает как прошло
- Ответ идёт в память и Identity Model

## Новые фичи (сессия 22 июня 2026)

### Overview — редизайн плашек
- Убрать одну большую синюю плашку
- Три компактных по сферам — Финансы, Здоровье, Проекты
- Каждая говорит своё с реальными цифрами
- Быстрые кнопки `[Да]` `[Нет]` `[Расскажу]` `[Пропустить]`
- Кнопки логируют ответ или открывают чат с контекстом

### Global Session Toggle
- Тоггл активного проекта из любого места (шапка, чат, Overview)
- Нажал → сессия пошла
- Остановил → AIR4 спрашивает что сделал
- Активная сессия идёт в контекст всех запросов

### Страница "Что AIR4 знает обо мне"
- Живой портрет — факты, паттерны, финансы, здоровье
- Видишь что знает неправильно — правишь через чат

### Финансовый календарь
- Техосмотр, страховка, разовые события через чат
- Тип `one_time` в таблице obligations
- Предупреждение за 7 дней

### Еженедельный разбор полётов
- План vs реальность по проектам
- Траты vs норма
- Тренировки vs прошлая неделя
- Честный короткий вывод

### Помощь выбраться из финансовой ямы
- Не просто анализ — конкретный план с цифрами
- Где деньги есть без боли, где сложнее

### Быстрый capture
- `Cmd+Shift+Space` — окошко поверх всего
- Одна фраза → Enter → закрылось
- Пишет в SQLite на маке

### Погода контекстно (Phase 10)
- Не виджет — ответ на конкретный вопрос
- "Катит на байке?" — знает байк, знает твой порог температуры

### Голос (после Phase 10)
- Whisper — распознавание, локально
- Kokoro/Piper — синтез, локально
- Утренний briefing голосом
