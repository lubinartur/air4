# AIR4 Brain Architecture

AIR4 reasoning combines several layers.

Input
↓
Context Builder
↓
Memory Retrieval
↓
Meaning Layer
↓
Reasoning Engine
↓
Response Generation

## Context Builder
Selects relevant events and meanings.

## Memory Retrieval
Uses embeddings to fetch semantically related events.

## Meaning Layer
Provides hypotheses and confirmed patterns.

## Reasoning Engine
LLM processes structured context.

## Output
AIR4 generates advice, analysis, or observation.
